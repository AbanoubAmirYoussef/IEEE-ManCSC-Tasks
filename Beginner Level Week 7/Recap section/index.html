Q1: Explain the four types of function binding in JavaScript: implicit binding, explicit binding, new binding, and default binding. How do they differ from each other?

Answer:
-----------
In JavaScript, this keyword binding happens in four main ways:
1- Implicit Binding – Occurs when a function is called as a method of an object. The this keyword refers to the object before the dot.
2- Explicit Binding – Done using .call(), .apply(), or .bind() to explicitly set the value of this.
3- New Binding – Happens when a function is called with the new keyword, creating a new object and setting this to it.
4- Default Binding – Happens when a function is called without any context; this defaults to the global object (window in browsers) or undefined in strict mode.

Q3: what the modules and what the differences between CommonJS modules and EJS modules?

Answer:
-----------
CommonJS:
1- Uses require() to import.
2- Uses module.exports to export.
3- Loads modules synchronously.
ESM:
1- Uses import statement.
2- Uses export statement.
3- Loads modules asynchronously.

Q4: What is the purpose of "scripts" attribute in package.json?

Answer:
-----------
The scripts in package.json defines custom commands that can be run with npm run. It is used to automate tasks like starting the server, running tests.
