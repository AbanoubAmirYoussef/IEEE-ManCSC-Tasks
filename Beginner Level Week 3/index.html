Theoretical:

Q1: Compare between the three types of quantifiers in regular expressions (?, +, *).

Answer:
-----------
? : Matches 0 or 1.
+ : Matches 1 or more.
* : Matches 0 or more.

Q2: What are the characters that can be represented by \w and \b in a regular expression?

Answer:
-----------
\w → matches any word character [A-Za-z0-9]
\b → word boundary (the start or end of a word)

Q3: Compare between the Map and WeakMap according to the way they save keys and values.

Answer:
-----------
Map:
- Keys can be any type (string, object, etc...)
- Iterable (Can use ForEach/Loop)
- Not GC-Friendly

WeakMap:
- Keys must be objects
- Not iterable
- Automatically removes entries if the object is no longer referenced (GC-Friendly)
