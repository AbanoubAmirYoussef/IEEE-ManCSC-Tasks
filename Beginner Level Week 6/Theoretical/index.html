Q1: According to your understanding; explain what's the difference between fs.readFile and fs.readFileSync methods.

Answer:
-----------
fs.readFile: Asynchronous, meaning it doesn't block the execution of the next lines of code need a callback to handle the result.
fs.readFileSync: synchronous, meaning it blocks the code execution until the file is read.

Q2: Explain with examples the difference between installing a package as a normal dependency vs. as a development dependency with npm.

Answer:
-----------
Normal dependencies (npm install package): required to run the app.
Development dependencies (npm install package --save-dev): only needed while developing, The app can run without it.

Q3: What is the purpose of the node_modules folder, and why should it typically be excluded from version control systems like Git?

Answer:
-----------
The node_modules folder contains all installed packages for your project. It should be excluded from Git (via .gitignore) because: It can be huge in size All packages are listed in package.json, so anyone can run npm install to get them again.
